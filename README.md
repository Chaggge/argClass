# ArgClass: Классификация аргументации в полемических текстах

Репозиторий содержит код для автоматической классификации аргументации в русскоязычных полемических текстах на основе таксономии Д. Уолтона. Реализована двухэтапная классификация: бинарная (атакующие vs. поддерживающие) и многоклассовая (типы атакующих и конфликтных схем). Используются BERT-подобные модели и малые LLM.

## Структура
ArgClass   
├── PreprocessData/ Парсинг JSON → CSV 
├── BERT/ Классификация BERT-моделями   
├── LLM/ Классификация LLM (few-shot промптинг)
